EXAMINE_CHECK_NONE-- Defines for the unique-examine element. -- Displays the description regardless. Pass no requirements.
EXAMINE_CHECK_MINDSHIELDFor displaying info to those with a mindshield implant. Pass no requirements.
EXAMINE_CHECK_ANTAGFor displaying info to those with a certain antag datum, e.g. "traitors". Pass a list of /datum/antagonist typepaths. You can optionally pass a special_affiliation with your element to overide the displayed antagonist name (Such as passing "Special agent" so it shows that instead of "Traitor")
EXAMINE_CHECK_JOBFor displaying info to specific jobs, e.g. "scientists". Pass a list of /datum/job typepaths.
EXAMINE_CHECK_DEPARTMENTFor displaying info to people a part of specific departments, e.g. "all jobs in service". Pass a bitflag of departments.
EXAMINE_CHECK_FACTIONFor displaying info to mob factions, e.g. "syndicate". Pass a string "faction" Note: factions aren't set very consistently, and often are just set by antag datums, so you should probably try to use other checks if possible (like antag datums).
EXAMINE_CHECK_SKILLCHIPFor displaying info to people with certain skill-chips. Pass a list of /obj/item/skillchip typepaths.
EXAMINE_CHECK_TRAITFor displayind info to people with certain traits. Pass a list of trait keys / strings.
EXAMINE_CHECK_SPECIESFor displayind info to people of certain species. Pass a list of /datum/species typepaths.
